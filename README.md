# Upload Download Express

Upload & Download files via API using Express with Typescript

### Prerequisites

```
Nodejs
```

### Installing

```
npm install
```

For Test (port 3000)

```
npm run start
```

Or Development (port 3100)

```
npm run development
```

Open file

```
index.html
```
